# calc-rust
素朴なパーサコンビネータによる電卓

# 仕様
* 入力はファイルから(hoge.txt)
* 正数(0-9)と(, ), +, -, *, /, %を読み取る。演算子の意味は一般的なものに則る。
* 負数を使用できないが、途中計算や結果が負になることは認める
* 出力は標準出力を使う。
* PEGによる文法
```
N = {start, exp, sum, prod, val, num} 
Σ = {(, ), + , - , * , / , %, [0-9]} 
P = {
   start = exp \n / ε
   exp   = prod ((`+` / '-') prod)*,
   prod  = val  ((`*` / `/` / '%') val)*,
   val   = num / `(` exp `)`,
   num   = [1-9][0-9]*,
 }
```
